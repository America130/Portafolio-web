# Portafolio-web
Diseñar y desarrollar un sitio web personal tipo portafolio profesional, que refleje mi identidad, mis habilidades, mis logros y mi trayectoria académica y laboral, el portafolio debía ser funcional, visualmente atractivo, fácil de navegar y adaptable a distintos dispositivos (computadoras, tabletas y teléfonos móviles).
